# permission
权限管理相关文档

## 权限描述
权限对应的数字越小，权限等级越高

| 权限名称  |  数字表示  |       描述       |
|:-----:|:------:|:--------------:|
|  管理员  |   0    |      最高权限      |
|  客服   |   1    |  允许审核评论、处理举报   |
| 保留权限  |  2-5   | 这些数字被保留，方便系统升级 |
| 普通用户  |   6    |      默认权限      |
|  禁言   |   7    |     不能发表评论     |
|  封号   |   8    |      不能登录      |
